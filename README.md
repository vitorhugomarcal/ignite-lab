## Rodando o projeto

Clone the project and access the folder.

```bash
$ git clone git@github.com:vitorhugomarcal/ignite-lab.git
$ cd ignite-lab
```

Follow the steps below:

```bash
# Install the dependencies
$ npm install
# Start the project
$ npm run dev
```

The app will be available for access on your browser
